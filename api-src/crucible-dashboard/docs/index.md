# Dashboard (`crucible-dashboard`)

Read-only Streamlit; powers nousergon.ai (public) and console.nousergon.ai
(private, Cloudflare Access).

Source: [`nousergon/crucible-dashboard`](https://github.com/nousergon/crucible-dashboard).

(`dash-web`, `marketing-apex`, `marketing-dev`, `marketing-finance`,
`marketing-metron`, `marketing-telos`, `marketing-vires`) — these aren't
valid Python import names (hyphens) so mkdocstrings can't render them as
